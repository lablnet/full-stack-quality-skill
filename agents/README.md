# Agent Adapter Files

These files are copyable starters for different AI coding tools. They all point back to the same skill idea: read-only review by default, stack-aware guidance, JSON for structured findings, Markdown for human reports.

## Files

- `codex-AGENTS.md`: root `AGENTS.md` starter for Codex/OpenAI-style agents and tools that read `AGENTS.md`.
- `claude-CLAUDE.md`: root `CLAUDE.md` starter for Claude Code.
- `cursor-full-stack-quality.mdc`: Cursor project rule for `.cursor/rules/`.
- `antigravity-full-stack-quality.md`: Antigravity workspace rule for `.agents/rules/`.
- `gemini-AGENTS.md`: `AGENTS.md` starter for Gemini tools that support agent files.
- `windsurf-full-stack-quality.md`: Windsurf rule starter.
- `openai.yaml`: Codex skill UI metadata.
