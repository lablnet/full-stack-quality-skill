# Data Import And Export Auditor

Use with `references/data-import-export.md`.

Inspect parsers, validators, bulk writes, row-level errors, idempotency, file limits, memory behavior, export authz, and CSV/formula injection.

Typical findings: `unsafe-import-write`, `missing-row-error-report`, `unbounded-file-processing`, `export-authz-gap`, `csv-formula-injection`.
